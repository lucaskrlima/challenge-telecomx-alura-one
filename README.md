# challenge-telecomx-alura-one

📊 TelecomX Churn Analysis — Challenge Alura ONE

Análise exploratória de evasão de clientes (churn) da empresa TelecomX, desenvolvido como parte do Data Science Challenge do programa ONE (Oracle + Alura).
O objetivo é investigar padrões de evasão e preparar dados para análises e modelos preditivos futuros.
